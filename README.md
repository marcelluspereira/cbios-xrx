# C-BIOS for XRX

Welcome to the **C-BIOS-XRX** repository. This project is a specialized fork of **[C-BIOS](https://www.google.com/search?q=http://cbios.sourceforge.net/)**, an open-source MSX BIOS compatible replacement.

### 🚀 Project Goal: Kickstarting XRX - eXperimental Research X

The primary objective of this fork is to adapt and utilize C-BIOS as the initial firmware kickstart for the **XRX prototype**.

By leveraging the stable, open-source framework of C-BIOS, we can rapidly bring up the core system architecture of the XRX hardware without relying on proprietary, copyrighted system ROMs. This fork serves as the foundational software bridge, allowing the XRX prototype to boot, initialize hardware, and execute code in a clean, compatible environment.

### 🌐 Learn More About XRX

The XRX (Experimental Research X) is a broader hardware and emulation development journey. To see how this repository fits into the overall hardware architecture, check out the main project page:

👉 **[Follow the XRX Project on Hackaday.io](https://hackaday.io/project/205929-xrx-experimental-research-x)**
